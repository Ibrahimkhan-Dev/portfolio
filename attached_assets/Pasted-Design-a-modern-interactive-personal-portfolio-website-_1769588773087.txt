Design a modern interactive personal portfolio website with a premium dark theme. The UI should use a deep black background, soft white text, and neon yellow accents for interactions. The overall style must feel futuristic, clean, minimal, and high-end, with smooth animations and optional 3D motion effects.

🎨 Color Palette (Strict)

Primary Background: #0B0B0F (deep black)

Secondary Background: #111118 (for sections/cards)

Primary Text: #F5F5F5 (soft white)

Secondary Text: #A0A0A0 (muted gray)

Accent / Interactive Yellow: #FFD400

Border / Line Color: rgba(255,255,255,0.08)

Glow Effect: yellow glow with low opacity on hover

✍ Typography (Modern + Clean)

Headings: Poppins / Space Grotesk / Sora (bold)

Body Text: Inter / Manrope (regular)

Use clear hierarchy:

H1 = large bold hero headline

H2 = section titles

Body = readable spacing with soft contrast

🌟 Core Design Style

Minimal layout but visually powerful

Use plenty of spacing (clean UI)

Smooth rounded corners (12px–18px)

Subtle glass effect on cards (optional)

Thin lines, soft shadows, modern grid alignment

Yellow used only for highlights, hover states, and key CTA buttons

⚡ Motion + Interaction Requirements

The website must feel interactive and premium:

Cursor & Hover

Yellow glow cursor trail OR subtle cursor-follow spotlight

Buttons with animated underline or border fill

Cards with 3D tilt on hover

Icons glow yellow on hover

Page Transitions

Smooth transitions between pages:

fade + slide

scale-in effect

section reveal animation

Navigation should feel smooth and animated

Scroll Animations

Sections reveal on scroll (fade-up + blur-to-clear)

Timeline animations for experience/education

Project cards appear with staggered animation

Optional 3D Elements (if possible)

3D hero background (floating shapes, sphere, cube, or particles)

Smooth rotation on scroll or mouse move

3D object should be subtle, not distracting
