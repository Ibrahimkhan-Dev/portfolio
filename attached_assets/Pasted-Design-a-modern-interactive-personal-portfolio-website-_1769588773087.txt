This is a React + TypeScript + Vite portfolio app with:

React 19

Vite 7

TypeScript strict mode

Tailwind CSS v4

Framer Motion for animation

Wouter for routing

Radix UI primitives for some UI elements

Architecturally, it is a single-page portfolio with one detail route:

/ → main portfolio landing page

/project/:id → project detail page

The layout is clean and modular:

components/sections/* → page sections like Hero, Projects, Experience, Contact

components/ui/* → reusable UI and animation primitives

pages/* → route-level pages

data/portfolio.ts → portfolio content/data source

So the project is not just “styled components”; it is organized with a clear separation between:

content/data

page sections

shared UI primitives

animation system
